To run the Selenium .SIDE file:
- Ensure you have Selenium IDE extension installed on Firefox: https://www.seleniumhq.org/selenium-ide/docs/en/introduction/getting-started/
- Once installed, restart your browser
- Click the Se icon for the Selenium Extension
- Choose "Open Existing Project"
- Select the "DeCoste-Tech-Interview.side" file
- Click the "Run Current Test" button. The 2nd button from the left that is directly next to the play button with 3 stacked lines next to it


To run the Java file w/ Maven:
- Ensure you have the GeckoDriver installed and added to your system environment 
-- File: https://github.com/mozilla/geckodriver/releases
-- Directions: https://www.softwaretestinghelp.com/geckodriver-selenium-tutorial/
- Ensure you have Maven installed and added to your system environment
-- File: https://maven.apache.org/download.cgi
-- Directions: http://maven.apache.org/install.html
- Open your IDE
- Pull project from GitHub
- Main project code located in src/main/java/org/example
- Open "DeCoste-Tech-Interview" project
- Select Run -> Run 'DeCosteSTETest'
